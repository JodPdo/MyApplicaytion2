# 📱 Business Card App

A modern Android Business Card application built with **Kotlin** and **Jetpack Compose**.  
This project demonstrates clean UI development using reusable composables and modern Android best practices.

---

## 🚀 Overview

The application displays a digital business card containing:

- Profile / Logo image
- Full name
- Professional title
- Phone number
- Social profile
- Email address

This project was created to practice Android UI development with Jetpack Compose.

---

## 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Android Studio
- Gradle

---

## 📚 Skills Demonstrated

### UI Development

- Declarative UI with Jetpack Compose
- Reusable composable functions
- Layout design using `Column`, `Row`, and `Spacer`

### Styling

- Typography customization
- Color system
- Alignment and spacing using `Modifier`

### Project Structure

- Clean composable separation
- Readable code organization

Requirements

Before running the project, make sure you have:

Clone this repository

Open Android Studio
Go to:
File > Open
Select the project folder
Wait for Gradle Sync to complete
Run the project:
Click Run ▶️ button

or

Press:
Shift + F10
Choose one device:
Android Emulator

or

Physical Device (USB Debugging enabled)
